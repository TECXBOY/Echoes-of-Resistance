choes of Resistance — Sierra Leone

**Echoes of Resistance** is a cinematic, single-page digital memorial to Sierra Leone's history of colonial resistance (1787–1961). It is designed to feel like a museum exhibit crossed with a film — with a loading sequence, scroll-based animations, an interactive map, a chronological timeline, letters, key figures, legacy cards, and a reflection wall.

This version is a **static HTML build** of the experience, intended for simple deployment (e.g. via Lovable, Netlify, Vercel, or any static host).

## Tech & Features

- Single `index.html` file
- Custom CSS for:
  - Loader screen and scroll progress bar
  - Hero section with Ken Burns-style background motion
  - Interactive dual map (pre‑colonial kingdoms vs colonial borders)
  - Timeline, letters, figures, legacy, reflections, and closing proverb
- All images embedded as base64, so no external assets are required

## Running Locally

You can open the file directly in a browser:

- Double-click `index.html`  
  or  
- Serve it with any static server:

npx serve .
